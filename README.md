u12nodpls
================

Shippable CI image for node.js on ubuntu 12.04 with commonly used services. Available versions are:

1. 0.10
2. 0.12
3. 4.23

**Services:**

1. elasticsearch 1.5 
2. memcached 1.4 
3. mongodb 3.0 
4. mysql 5.6 
5. postgres 9.4 
6. rabbitmq 3.5 
7. redis 3.0 
8. selenium 2.45 
9. sqllite 3
